# Enhanced OSINT Security Assessment Tool

A Python-based tool for automated OSINT analysis and vulnerability threat scoring.

## Features

- **Multi-source OSINT**: DNS, WHOIS, ports, SSL/TLS, threat intelligence APIs
- **Email Fraud Detection**: Identifies phishing indicators and suspicious infrastructure
- **Vulnerability Correlation**: Detects compound patterns across data sources
- **Threat Scoring**: CVSS-inspired 0–10 scoring system
- **Executive Summaries**: NLG-based business-readable reports
- **Docker Support**: Containerised for secure deployment

## Requirements

- Python 3.8+
- Docker (optional)

## Installation

```bash
git clone https://github.com/YOUR-USERNAME/OSINT-Security-Tool.git
cd OSINT-Security-Tool
pip install -r requirements.txt
```

## Usage

```bash
python osint_tool.py --target example.com
```

## Example Output

Generates threat reports with CVSS-style scoring, email fraud detection, and executive summary.
